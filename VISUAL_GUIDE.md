## Color Schemes

### Student Interface
- **Primary:** Blue (#3B82F6)
- **Background:** Light blue gradients
- **Accent:** Cyan
- **Purpose:** Trustworthy, calm, educational

### Admin Interface
- **Primary:** Purple (#9333EA)
- **Background:** Purple to indigo gradients
- **Accent:** Deep purple
- **Purpose:** Professional, authoritative

### Success States
- **Primary:** Green (#10B981)
- **Background:** Green to teal gradients
- **Purpose:** Positive confirmation

### Alerts
- **Info:** Blue (#3B82F6)
- **Warning:** Yellow (#F59E0B)
- **Error:** Red (#EF4444)
- **Success:** Green (#10B981)

## Interactive Elements

### Buttons
- **Primary:** Solid color with hover effect
- **Secondary:** Outlined with hover fill
- **Disabled:** Gray with no pointer
- **Hover:** Slight lift with shadow increase

### Forms
- **Focus:** Blue ring around input
- **Error:** Red border with message
- **Valid:** Normal border
- **Placeholder:** Gray text

### Cards
- **Hover:** Slight lift and shadow increase
- **Active:** Pressed down effect
- **Border:** Subtle gray or colored

## Typography

### Headings
- **H1:** 3xl, bold (Landing page titles)
- **H2:** 2xl, bold (Section headers)
- **H3:** xl, semibold (Card titles)

### Body Text
- **Regular:** base, normal (Body content)
- **Small:** sm, normal (Helper text)
- **Tiny:** xs, normal (Fine print)

### Special Text
- **Labels:** sm, medium (Form labels)
- **Buttons:** base, semibold (Button text)
- **Links:** sm, normal with underline on hover
