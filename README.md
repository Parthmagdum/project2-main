### How to Use

1. **Student Login**: Enter your Student ID to access your dashboard
2. **View All Feedback**: See complete history with "Total: X feedback" counter
3. **Submit Feedback**: Click "Submit New Feedback" button
4. **Filter Feedback**: Use dropdowns to filter by sentiment or category
5. **Reply to Faculty**: Click "Reply to Faculty" on any feedback with a response
